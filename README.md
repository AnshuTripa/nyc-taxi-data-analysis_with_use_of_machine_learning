# 🚖 NYC Taxi Data Analysis & Prediction App

## 📌 Overview
This project is a **Streamlit-based web application** that performs data analysis on NYC taxi trip data and predicts trip distance using a **Machine Learning model (Random Forest Regressor)**.

The application provides an interactive interface where users can explore data, visualize patterns, and train a model with customizable parameters.

## 🎯 Key Features
- 📊 Data visualization (Pickup location distribution)
- 🔍 Interactive feature selection
- ⚙️ Adjustable model parameters (max_depth, n_estimators)
- 🤖 Machine Learning model (Random Forest)
- 📈 Performance metrics:
  - Mean Absolute Error (MAE)
  - Mean Squared Error (MSE)
  - R² Score
- 🌐 Fully interactive UI using Streamlit

## ⚙️ How to Run Locally

# Clone repository
git clone https://github.com/yourusername/nyc-taxi-data-analysis_with_use_of_machine_learning.git

# Go to project folder
cd nyc-taxi-data-analysis_with_use_of_machine_learning

# Install dependencies
pip install -r requirements.txt

# Run the app
streamlit run app.py
